# honey-bee — Nutzwertanalyse & Anbietervergleich

Web-App zur Erstellung von Nutzwertanalysen und Anbietervergleichen im Projektkontext.

## Features

- Kriterien definieren und verwalten
- Paarweiser Vergleich zur Gewichtung der Kriterien
- Anbieter bewerten (Skala: 0–10)
- Gewichtete Auswertung mit Ranking und Empfehlung
- Netzdiagramm (Radar Chart) zur Visualisierung
- Projekt-Import / Export (JSON)

## Entwicklung

```bash
npm install
npm run dev
```

## Build & Deployment

```bash
npm run build
```

Deployment via GitHub Pages (siehe `.github/workflows/deploy.yml`).
